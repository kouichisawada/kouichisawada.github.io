勉強したこと

hugoの使い方 /site-demo内で $ hugo new post/ファイル名.md このコマンドでcontent内postにmdファイルができる。 hugoコマンドを打つ時は、/site-demo内で行う。 そうすると、/content/post/ファイル名.mdが変換されて、/public/post内にindex.htmlファイルが生成される。 git コマンド $ git add . を/publicにいる時に行うと、ステージにのる。 その状態で $ git commit -m “コメント”　を打つと、logに残る。 これはあとで見返すことができる。 そして $ git push すると、リポジトリ内に自分の変更した内容などがpushされる。 etc あとは様々な仕様や、どういう仕組みでサイトが出来上がっているかなど、教えてもらった。
